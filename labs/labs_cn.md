## 实验准备 (课前需做) 

**课前必做** :
1. 将电脑 Windows 11 专业版升级到最新版本 ( **由于家庭版会导致某些实验出现异常**, 
**请大家通过 my.whu.edu.cn 的正版软件版块安装专业版的 win11** )

2. 将电脑设置为开发者模式

3. 安装 Visual Studio Community 2022 及相关插件, 参见教学幻灯片

4. 通过 my.whu.edu.cn 的正版软件版块安装专业版的 Office （如果用家庭版会导致 COM 相关实验无法进行）

Windows 应用开发环境安装参见 [Install tools for Windows app development](https://learn.microsoft.com/en-us/windows/apps/windows-app-sdk/set-up-your-development-environment)
